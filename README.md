
# Streaming Price

## Instalación 

Instalar el proyecto con npm

```bash
  cd Streaming Price
  npm install 
```
    
## Ejecutar Localmente

Para implementar este proyecto, ejecute

```bash
  cd Streaming Price
  npm run dev
```
