- Css vanilla 

## Ejecutar Localmente

Clona el repositorio

```bash
  git clone https://github.com/PANCHOv96/StreamingPrice
```

Ir al directorio del proyecto

```bash
  cd StreamingPrice
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run dev
```
